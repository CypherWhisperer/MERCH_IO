## Stack
- As of originally writting this (_November 5th, 2025_) the Project leverages:
  - **Vite** - build tool
  - **React** - Frontend UI library:
    - ***Dependencies***
      - Framer Motion (_motion_) - *for animation*
      - React Router Dom - *for internal routing*
      - React icons - *for ease of working with icon packs*
      - Tailwind CSS - *for styling simple components and layout*
      - SASS - *for general styling via SCSS modules*
      - POST CSS - 

- Ultimately, the project will evolve to be a NEXT js project - being as it leverages React as its front end UI library
